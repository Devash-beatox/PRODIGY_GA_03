# PRODIGY_GA_03 — Text Generation with Markov Chains

## Task
Implemented a simple text generation algorithm using Markov chains — a
statistical model that predicts the next word based on the previous word(s),
without any neural network or training.

## What I learned
- How Markov chains work: counting which words follow which, building a
  probability lookup table from source text
- The difference between order-1 and order-2 chains, and the coherence
  vs. randomness trade-off between them
- Why this approach was how text generation worked before deep learning,
  and how it compares to GPT-2's attention-based approach (Task 01)
